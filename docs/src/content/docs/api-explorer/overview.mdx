The API Explorer provides a built-in tool for exploring, testing, and learning the LogicMonitor REST API—without needing API tokens or external tools.

## What Is the API Explorer?

The API Explorer is an integrated REST client that:

<CardGrid>
  <FeatureCard title="Session Auth" icon="check-circle" color="green">
    Uses your active LogicMonitor session—no API tokens needed.
  </FeatureCard>
  <FeatureCard title="Endpoint Catalog" icon="book" color="blue">
    Browse all available API endpoints organized by category.
  </FeatureCard>
  <FeatureCard title="Request Builder" icon="edit" color="purple">
    Build and execute requests with guided parameter input.
  </FeatureCard>
  <FeatureCard title="Request History" icon="history" color="orange">
    Review past requests and reload them for quick retesting.
  </FeatureCard>
</CardGrid>

## Opening the API Explorer

<CardGrid>
  <FeatureCard title="Welcome Screen" icon="rocket" color="orange">
    Click **API Explorer** on the welcome screen.

### Main Panel

| Section | Description |
|---------|-------------|
| **Method Selector** | Choose HTTP method (GET, POST, PUT, PATCH, DELETE) |
| **Endpoint Field** | Enter or build the API path |
| **Request Body** | JSON body for POST/PUT/PATCH requests |
| **Response Viewer** | Formatted API response with syntax highlighting |

### Right Sidebar

The sidebar provides three tabs:

- **Endpoints**: Browse all available API endpoints
- **Variables**: Manage reusable environment variables
- **History**: View and reload past API requests

## Key Features

### Session-Based Authentication

Unlike traditional API tools, you don't need API tokens:

<Steps>

1. Log into LogicMonitor in your browser

2. Open LMDA Composer—it uses your session cookie

3. All API calls execute with your permissions automatically

</Steps>

<Aside type="tip" title="No Token Management">
  No need to create, store, or rotate API tokens. Your session handles authentication automatically. Just stay logged into LogicMonitor.
</Aside>

### Endpoint Discovery

Browse the complete LogicMonitor API:

- Organized by resource category (Devices, Alerts, Settings, etc.)
- Search across all endpoints
- View parameter requirements
- See example requests and response schemas

### Request Building

Build requests with guidance:

- Auto-complete for path parameters
- Query parameter helpers with validation
- JSON body templates
- Pre-send request validation

### Response Formatting

Responses display clearly:

- Syntax-highlighted JSON
- Collapsible sections for large responses
- Copy functionality
- Error highlighting for failed requests

## Quick Start Example

Test your connection with a simple API call:

<Steps>

1. Open API Explorer

2. Method is <HttpMethodBadge method="GET" size="sm" /> (default)

3. Enter `/setting/version` as the endpoint

4. Click **Send Request**

5. View the version response showing your portal's API version

</Steps>

## Permissions

API calls execute with your user permissions:

- Only see data you have access to
- Can only modify resources you can edit
- Admin endpoints require admin permissions

<Aside type="note" title="Permission Errors">
  A **403 Forbidden** error means you don't have permission for that endpoint. Check your LogicMonitor user role in Settings → Users and Roles.
</Aside>

---

## Rate Limiting

LogicMonitor applies rate limits to API calls:

- Standard rate limits apply to all requests
- Response headers show remaining quota
- **429 Too Many Requests** indicates rate limiting

The status bar shows response timing to help identify slow endpoints.
