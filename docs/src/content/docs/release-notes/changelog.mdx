## v1.5.0 — Create LogicModule Wizard

<div style="display: flex; align-items: center; gap: 0.75rem; margin: 1rem 0;">
  <Badge text="v1.5.0" variant="note" />
  <span style="font-size: 0.875rem; color: var(--sl-color-gray-3);">January 2026</span>
</div>

This release introduces a powerful wizard for creating new LogicModules directly from LMDA Composer, plus enhanced datapoint editing features.

### Create LogicModule Wizard

<CardGrid>
  <FeatureCard title="Multi-Step Wizard" icon="rocket" color="blue">
    Guided workflow for creating new modules with type selection, naming, and script configuration.
  </FeatureCard>
  <FeatureCard title="All Module Types" icon="layers" color="purple">
    Support for DataSource, ConfigSource, TopologySource, PropertySource, LogSource, EventSource, and DiagnosticSource.
  </FeatureCard>
  <FeatureCard title="Smart Defaults" icon="check-circle" color="green">
    Auto-configures required fields like datapoints, config checks, and alert settings per module type.
  </FeatureCard>
  <FeatureCard title="Safe Development" icon="lock" color="orange">
    Sets AppliesTo to `false()` so new modules are inactive until you configure targeting.
  </FeatureCard>
</CardGrid>

### Module Type Support

| Type | Display Name | Collect Interval | AD Support | Default Requirement |
|------|-------------|------------------|------------|---------------------|
| DataSource | Yes | 5 min | Yes | exitCode datapoint |
| ConfigSource | Yes | 1 hour | Yes | RetrievalCheck |
| TopologySource | No | 1 hour | No | None |
| PropertySource | No | On discovery | No | None |
| LogSource | No | 5 min | No | logFields, resourceMapping |
| EventSource | No | 30 min | No | Alert settings |
| DiagnosticSource | No | On-demand | No | None |

### Status Display Names

<CardGrid>
  <FeatureCard title="Value Translation" icon="translate" color="cyan">
    Translate numeric datapoint values to human-readable status strings in the LogicMonitor UI.
  </FeatureCard>
  <FeatureCard title="Flexible Operators" icon="filter" color="purple">
    Configure conditions with equals, greater than, less than, and range operators.
  </FeatureCard>
  <FeatureCard title="Multiple Translations" icon="list" color="blue">
    Add multiple status translations per datapoint for different value ranges.
  </FeatureCard>
</CardGrid>

Example: Display `1` as "Online" and `0` as "Offline" instead of raw numbers.

### Alert Message Templates

<CardGrid>
  <FeatureCard title="Token Support" icon="code" color="green">
    Use LogicMonitor tokens in alert subject and body templates for dynamic messages.
  </FeatureCard>
  <FeatureCard title="Custom Messages" icon="edit" color="orange">
    Customize alert notifications with context-specific information.
  </FeatureCard>
</CardGrid>

### API Explorer Enhancements

<CardGrid>
  <FeatureCard title="Docs Tab" icon="book" color="blue">
    New sidebar tab showing endpoint documentation including parameters, schemas, and examples.
  </FeatureCard>
  <FeatureCard title="Schema Properties" icon="list" color="purple">
    View request body schema with nested property details and required field indicators.
  </FeatureCard>
  <FeatureCard title="Example Generation" icon="code" color="green">
    Auto-generated example request bodies based on the endpoint schema.
  </FeatureCard>
</CardGrid>

### UI Enhancements

<CardGrid>
  <FeatureCard title="Auto-Populate Display Name" icon="pencil" color="blue">
    Display name automatically syncs with module name until manually edited.
  </FeatureCard>
  <FeatureCard title="Confirmation Review" icon="document" color="purple">
    Review all wizard choices before creating the module in your portal.
  </FeatureCard>
</CardGrid>

## v1.4.0 — Datapoints & Config Checks Editing

<div style="display: flex; align-items: center; gap: 0.75rem; margin: 1rem 0;">
  <Badge text="v1.4.0" variant="note" />
  <span style="font-size: 0.875rem; color: var(--sl-color-gray-3);">January 2026</span>
</div>

This release adds full editing capabilities for DataSource datapoints and ConfigSource config checks directly within LMDA Composer.

### Datapoint Editing

<CardGrid>
  <FeatureCard title="Full CRUD Operations" icon="edit" color="blue">
    Add, edit, and delete datapoints for DataSource modules without leaving the editor.
  </FeatureCard>
  <FeatureCard title="Normal & Complex Types" icon="layers" color="purple">
    Support for both standard collection datapoints and expression-based complex datapoints.
  </FeatureCard>
  <FeatureCard title="Alert Thresholds" icon="bell" color="orange">
    Configure Warning, Error, and Critical thresholds with visual severity indicators.
  </FeatureCard>
  <FeatureCard title="Rich Table View" icon="document" color="green">
    Expandable rows showing description, alert messages, and no-data settings.
  </FeatureCard>
</CardGrid>

### Config Check Editing

<CardGrid>
  <FeatureCard title="Five Check Types" icon="check-circle" color="blue">
    Any Change, Groovy Script, Fetch Failure, Missing Field, and Value Check.
  </FeatureCard>
  <FeatureCard title="Change Exclusions" icon="filter" color="purple">
    Multi-value inputs for regex patterns, line prefixes, and contains strings.
  </FeatureCard>
  <FeatureCard title="Groovy Script Editor" icon="code" color="green">
    Full script editing for custom validation logic.
  </FeatureCard>
  <FeatureCard title="Alert Configuration" icon="settings" color="orange">
    Severity levels, clear on acknowledgement, and auto-clear intervals.
  </FeatureCard>
</CardGrid>

### UI Enhancements

<CardGrid>
  <FeatureCard title="Severity Icons" icon="warning" color="yellow">
    Distinct icons for Warning (circle), Error (triangle), and Critical (octagon) alerts.
  </FeatureCard>
  <FeatureCard title="Threshold Badges" icon="tag" color="cyan">
    Color-coded alert threshold display matching LogicMonitor's visual style.
  </FeatureCard>
  <FeatureCard title="Change Indicators" icon="git-branch" color="green">
    Visual indicators for added, modified, and removed items in the commit dialog.
  </FeatureCard>
  <FeatureCard title="Improved Diffing" icon="code" color="purple">
    Pre-sorted arrays and consistent object key ordering for accurate change detection.
  </FeatureCard>
</CardGrid>

### Keyboard & Navigation

<CardGrid>
  <FeatureCard title="Shortcut Chords" icon="keyboard" color="blue">
    Cmd+K chord pattern avoids browser extension keyboard conflicts.
  </FeatureCard>
  <FeatureCard title="Empty State Status Bar" icon="information" color="purple">
    Shows portal context and connection status when no tabs are open.
  </FeatureCard>
  <FeatureCard title="Documentation Links" icon="book" color="green">
    In-app links to full documentation from the status bar help popup.
  </FeatureCard>
</CardGrid>

### File Management

<CardGrid>
  <FeatureCard title="Open Module Directory" icon="folder" color="blue">
    Open saved module folders directly with Cmd+K, F to restore module binding.
  </FeatureCard>
  <FeatureCard title="Save Modules to Disk" icon="save" color="green">
    Portal-bound modules can be saved with metadata (module.json) for offline work.
  </FeatureCard>
  <FeatureCard title="Git-Friendly Workflow" icon="git-branch" color="purple">
    Module directory structure supports version control for team collaboration.
  </FeatureCard>
  <FeatureCard title="Duplicate Handle Fix" icon="check-circle" color="cyan">
    Fixed issues with duplicate file handles when reopening the same filename.
  </FeatureCard>
</CardGrid>

### API Explorer

<CardGrid>
  <FeatureCard title="API Welcome Screen" icon="rocket" color="orange">
    New welcome screen with quick actions for API exploration and endpoint discovery.
  </FeatureCard>
</CardGrid>

## v1.3.5 — Initial Public Release

<div style="display: flex; align-items: center; gap: 0.75rem; margin: 1rem 0;">
  <Badge text="v1.3.5" variant="success" />
  <span style="font-size: 0.875rem; color: var(--sl-color-gray-3);">January 2026</span>
</div>

LMDA Composer is now available for LogicMonitor power users! This release includes a comprehensive set of features for LogicModule development.

### Script Editor

<CardGrid>
  <FeatureCard title="Monaco Editor" icon="edit" color="blue">
    Full-featured code editor with syntax highlighting for Groovy and PowerShell.
  </FeatureCard>
  <FeatureCard title="Code Features" icon="code" color="purple">
    Code folding, find and replace, multiple cursors, line numbers.
  </FeatureCard>
  <FeatureCard title="Auto-Save" icon="check-circle" color="green">
    Automatic draft saving for crash recovery.
  </FeatureCard>
</CardGrid>

### Script Execution

<CardGrid>
  <FeatureCard title="Collector Execution" icon="play" color="green">
    Run scripts on any connected collector with real-time output streaming.
  </FeatureCard>
  <FeatureCard title="Four Execution Modes" icon="layers" color="purple">
    Freeform, Active Discovery, Collection, and Batch Collection modes.
  </FeatureCard>
  <FeatureCard title="Output Parsing" icon="document" color="blue">
    Parsed output for AD and Collection modes with validation.
  </FeatureCard>
</CardGrid>

### Portal Tools

<Tabs>
  <TabItem label="LogicModule Browser" icon="seti:folder">
    Browse and import scripts from your portal. Filter by module type and navigate the module hierarchy.
  </TabItem>
  <TabItem label="LogicModule Search" icon="magnifier">
    Search across all module scripts, datapoints, and metadata. Find code patterns across your entire portal.
  </TabItem>
  <TabItem label="AppliesTo Toolbox" icon="funnel">
    Test and validate AppliesTo expressions. See exactly which resources match your expression.
  </TabItem>
  <TabItem label="Debug Commands" icon="rocket">
    Run collector debug commands with formatted results. Includes comprehensive health check reports.
  </TabItem>
</Tabs>

### API Explorer

<CardGrid>
  <FeatureCard title="Session Auth" icon="check-circle" color="green">
    Uses your browser session—no API tokens needed.
  </FeatureCard>
  <FeatureCard title="Endpoint Catalog" icon="book" color="blue">
    Browse and search available LogicMonitor API endpoints.
  </FeatureCard>
  <FeatureCard title="Request Builder" icon="edit" color="purple">
    Build requests with query parameters, headers, and body.
  </FeatureCard>
  <FeatureCard title="Response Viewer" icon="braces" color="orange">
    Formatted JSON responses with syntax highlighting.
  </FeatureCard>
</CardGrid>

### Module Management

<CardGrid>
  <FeatureCard title="Module Details Editor" icon="settings" color="default">
    Edit module metadata including name, description, AppliesTo, and more.
  </FeatureCard>
  <FeatureCard title="Module Lineage" icon="git-branch" color="cyan">
    View Exchange release history for LogicMonitor core modules.
  </FeatureCard>
  <FeatureCard title="Commit" icon="upload" color="green">
    Push script and metadata changes back to LogicMonitor.
  </FeatureCard>
</CardGrid>

### Workspace Features

<CardGrid>
  <FeatureCard title="Multi-Tab Interface" icon="layers" color="blue">
    Work with multiple scripts simultaneously. Module binding indicators on tabs.
  </FeatureCard>
  <FeatureCard title="Local File Management" icon="folder" color="purple">
    Open, save, and manage local script files with File System Access API.
  </FeatureCard>
  <FeatureCard title="Snippets Library" icon="puzzle" color="orange">
    Built-in and custom code snippets for rapid development.
  </FeatureCard>
  <FeatureCard title="Execution History" icon="history" color="cyan">
    Track past executions with script replay capability.
  </FeatureCard>
</CardGrid>

### User Experience

<CardGrid>
  <FeatureCard title="Welcome Screen" icon="rocket" color="orange">
