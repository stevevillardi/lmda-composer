## v1.3.5 — Initial Public Release

<div style="display: flex; align-items: center; gap: 0.75rem; margin: 1rem 0;">
  <Badge text="v1.3.5" variant="success" />
  <span style="font-size: 0.875rem; color: var(--sl-color-gray-3);">January 2026</span>
</div>

LMDA Composer is now available for LogicMonitor power users! This release includes a comprehensive set of features for LogicModule development.

### Script Editor

<CardGrid>
  <FeatureCard title="Monaco Editor" icon="edit" color="blue">
    Full-featured code editor with syntax highlighting for Groovy and PowerShell.
  </FeatureCard>
  <FeatureCard title="Code Features" icon="code" color="purple">
    Code folding, find and replace, multiple cursors, line numbers.
  </FeatureCard>
  <FeatureCard title="Auto-Save" icon="check-circle" color="green">
    Automatic draft saving for crash recovery.
  </FeatureCard>
</CardGrid>

### Script Execution

<CardGrid>
  <FeatureCard title="Collector Execution" icon="play" color="green">
    Run scripts on any connected collector with real-time output streaming.
  </FeatureCard>
  <FeatureCard title="Four Execution Modes" icon="layers" color="purple">
    Freeform, Active Discovery, Collection, and Batch Collection modes.
  </FeatureCard>
  <FeatureCard title="Output Parsing" icon="document" color="blue">
    Parsed output for AD and Collection modes with validation.
  </FeatureCard>
</CardGrid>

### Portal Tools

<Tabs>
  <TabItem label="LogicModule Browser" icon="seti:folder">
    Browse and import scripts from your portal. Filter by module type and navigate the module hierarchy.
  </TabItem>
  <TabItem label="LogicModule Search" icon="magnifier">
    Search across all module scripts, datapoints, and metadata. Find code patterns across your entire portal.
  </TabItem>
  <TabItem label="AppliesTo Toolbox" icon="funnel">
    Test and validate AppliesTo expressions. See exactly which resources match your expression.
  </TabItem>
  <TabItem label="Debug Commands" icon="rocket">
    Run collector debug commands with formatted results. Includes comprehensive health check reports.
  </TabItem>
</Tabs>

### API Explorer

<CardGrid>
  <FeatureCard title="Session Auth" icon="check-circle" color="green">
    Uses your browser session—no API tokens needed.
  </FeatureCard>
  <FeatureCard title="Endpoint Catalog" icon="book" color="blue">
    Browse and search available LogicMonitor API endpoints.
  </FeatureCard>
  <FeatureCard title="Request Builder" icon="edit" color="purple">
    Build requests with query parameters, headers, and body.
  </FeatureCard>
  <FeatureCard title="Response Viewer" icon="braces" color="orange">
    Formatted JSON responses with syntax highlighting.
  </FeatureCard>
</CardGrid>

### Module Management

<CardGrid>
  <FeatureCard title="Module Details Editor" icon="settings" color="default">
    Edit module metadata including name, description, AppliesTo, and more.
  </FeatureCard>
  <FeatureCard title="Module Lineage" icon="git-branch" color="cyan">
    View Exchange release history for LogicMonitor core modules.
  </FeatureCard>
  <FeatureCard title="Commit" icon="upload" color="green">
    Push script and metadata changes back to LogicMonitor.
  </FeatureCard>
</CardGrid>

### Workspace Features

<CardGrid>
  <FeatureCard title="Multi-Tab Interface" icon="layers" color="blue">
    Work with multiple scripts simultaneously. Module binding indicators on tabs.
  </FeatureCard>
  <FeatureCard title="Local File Management" icon="folder" color="purple">
    Open, save, and manage local script files with File System Access API.
  </FeatureCard>
  <FeatureCard title="Snippets Library" icon="puzzle" color="orange">
    Built-in and custom code snippets for rapid development.
  </FeatureCard>
  <FeatureCard title="Execution History" icon="history" color="cyan">
    Track past executions with script replay capability.
  </FeatureCard>
</CardGrid>

### User Experience

<CardGrid>
  <FeatureCard title="Welcome Screen" icon="rocket" color="orange">
    Quick actions to get started—new file, open file, import module.
  </FeatureCard>
  <FeatureCard title="Command Palette" icon="search" color="purple">
    Keyboard-driven workflow with <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>.
  </FeatureCard>
  <FeatureCard title="Keyboard Shortcuts" icon="keyboard" color="blue">
    Comprehensive shortcuts for power users.
  </FeatureCard>
  <FeatureCard title="Customizable Settings" icon="settings" color="green">
    Theme, font size, editor preferences, and more.
  </FeatureCard>
</CardGrid>

### Known Limitations

<Aside type="note">
  - Extension not available in Firefox/Safari
  - PowerShell only runs on Windows collectors
  - Custom editor themes not yet supported
</Aside>
