LMDA Composer provides extensive keyboard shortcuts for efficient workflow. This page lists all available shortcuts.

<Aside type="tip" title="Chord Shortcuts">
  Many shortcuts use a **chord pattern**: press <kbd>Cmd</kbd>+<kbd>K</kbd> (Mac) or <kbd>Ctrl</kbd>+<kbd>K</kbd> (Windows/Linux), release, then press the second key. The status bar shows "⌘K waiting..." when a chord is pending.
</Aside>

## Global Shortcuts

These shortcuts work anywhere in LMDA Composer:

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Run Script / Send API Request | <kbd>Ctrl</kbd>+<kbd>Enter</kbd> | <kbd>Cmd</kbd>+<kbd>Enter</kbd> |
| Open Command Palette | <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> | <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> |
| Toggle Right Sidebar | <kbd>Ctrl</kbd>+<kbd>B</kbd> | <kbd>Cmd</kbd>+<kbd>B</kbd> |
| Open Settings | <kbd>Ctrl</kbd>+<kbd>,</kbd> | <kbd>Cmd</kbd>+<kbd>,</kbd> |

## Command Palette

When the Command Palette is open:

| Key | Action |
|-----|--------|
| <kbd>↑</kbd> / <kbd>↓</kbd> | Navigate commands |
| <kbd>Enter</kbd> | Execute command |
| <kbd>Escape</kbd> | Close palette |
| Type | Filter commands |

## Tips for Learning Shortcuts

- **Start with Essentials** — Most commonly used: <kbd>Cmd</kbd>+<kbd>Enter</kbd> (Run Script), <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> (Command Palette), <kbd>Cmd</kbd>+<kbd>S</kbd> (Save), <kbd>Cmd</kbd>+<kbd>B</kbd> (Toggle Sidebar)
- **Learn the Chord Pattern** — Press <kbd>Cmd</kbd>+<kbd>K</kbd>, watch for "waiting..." in status bar, then press the action key. Most portal tools and file operations use this pattern.
- **Use Command Palette** — The Command Palette shows shortcuts next to commands. Search for an action to see its shortcut.
