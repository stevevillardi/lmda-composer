LMDA Composer is a Chrome extension that provides a focused workspace for LogicMonitor script development. It works with Chrome, Edge, Brave, and other Chromium-based browsers.

## Prerequisites

Before installing, make sure you have:

<CardGrid>
  <FeatureCard title="Chromium Browser" icon="laptop" color="blue">
    Chrome, Edge, Brave, or another Chromium-based browser installed on your computer.
  </FeatureCard>
  <FeatureCard title="LogicMonitor Access" icon="globe" color="green">
    A LogicMonitor portal account with permission to run scripts on collectors.
  </FeatureCard>
</CardGrid>

## Installation Methods

Choose your preferred installation method:

<Tabs>
  <TabItem label="Chrome Web Store" icon="star">
    The easiest way to install LMDA Composer:

    1. Visit the [LMDA Composer Chrome Web Store page](https://chromewebstore.google.com/detail/bopacknfeabibohddhdakmlmfekijbea)
    2. Click **Add to Chrome**
    3. Click **Add extension** in the confirmation popup
    4. The LMDA Composer icon appears in your toolbar

    <Aside type="tip" title="Pin the Extension">
      Click the puzzle piece icon in your toolbar, then click the pin icon next to LMDA Composer for quick access.
    </Aside>
  </TabItem>
  
  <TabItem label="Developer Mode" icon="github">
    For the latest development version or to contribute:

    1. Clone the repository:
       ```bash
       git clone https://github.com/stevevillardi/lmda-composer.git
       cd lmda-composer
       ```

    2. Install dependencies and build:
       ```bash
       npm install
       npm run build
       ```

    3. Load in Chrome:
       - Navigate to `chrome://extensions`
       - Enable **Developer mode** (top right)
       - Click **Load unpacked**
       - Select the `dist` folder

    <Aside type="note">
      Developer builds don't auto-update. Pull the latest changes and rebuild to update.
    </Aside>
  </TabItem>
</Tabs>

## Verify Installation

After installing, you should see the LMDA Composer icon in your browser toolbar:

<img src="/lmda-composer/getting-started/installation-toolbar.png" alt="Browser toolbar showing the LMDA Composer extension icon" style="border-radius: 0.5rem; margin: 1.5rem 0;" />

Click the icon to open LMDA Composer in a new tab.

## Browser Compatibility

<BrowserCard name="Google Chrome" icon="seti:chrome" supported={true}>
  <p>Full support for all features including File System Access API.</p>
</BrowserCard>

<BrowserCard name="Microsoft Edge" icon="seti:edge" supported={true}>
  <p>Full support for all features. Works identically to Chrome.</p>
</BrowserCard>

<BrowserCard name="Brave" icon="seti:default" supported={true} note="Requires configuration">
  <p>Supported with additional setup. Enable File System Access API:</p>
  <ol>
    <li>Open <code>brave://flags</code> in a new tab</li>
    <li>Search for "File System Access API"</li>
    <li>Set it to <strong>Enabled</strong></li>
    <li>Restart Brave</li>
  </ol>
</BrowserCard>

<BrowserCard name="Vivaldi" icon="seti:default" supported={true} note="Requires configuration">
  <p>Supported with file access enabled:</p>
  <ol>
    <li>Open <code>vivaldi://extensions</code></li>
    <li>Select "LMDA Composer"</li>
    <li>Enable "Allow access to file URLs"</li>
  </ol>
</BrowserCard>

<BrowserCard name="Firefox / Safari" icon="seti:default" supported={false}>
  <p>Not supported at this time. Use Chrome or Edge for the best experience.</p>
</BrowserCard>
