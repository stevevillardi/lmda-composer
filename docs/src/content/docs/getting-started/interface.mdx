## Toolbar

The toolbar contains context selection and primary actions.

<img src="/lmda-composer/getting-started/interface-toolbar.png" alt="LMDA Composer toolbar" style="border-radius: 0.5rem; margin: 1rem 0;" />

### Context Selection
- **Portal** — Your LogicMonitor account (green indicator when connected)
- **Collector** — Where scripts execute (only online collectors available)
- **Device** — Target for property access (optional)

### Script Configuration
- **Language** — Toggle between Groovy and PowerShell
- **Mode** — Freeform, AD, Collection, or Batch

### Module Actions
- **Lineage** — View Exchange release history
- **Details** — Edit module metadata, datapoints, and settings
- **Commit** — Push changes back to LogicMonitor

### Execute
- **Run Script** — Execute on selected collector
- **Actions Menu** — Save, export, import options

---

## Tab Bar

Manage multiple open files and module scripts.

| Feature | Description |
|---------|-------------|
| **Multiple Tabs** | Work on multiple scripts simultaneously |
| **Dirty Indicator** | Dot appears on tabs with unsaved changes |
| **Language Badge** | <Badge text="GR" variant="success" /> for Groovy, <Badge text="PS" variant="note" /> for PowerShell |
| **Module Binding** | Link icon indicates tabs bound to a LogicModule |

### Tab Actions

Right-click any tab for additional options:
- Rename tab
- Close tab / Close others / Close all
- Copy tab name

---

## Code Editor

A professional editing experience powered by the Monaco editor.

| Feature | Description |
|---------|-------------|
| **Syntax Highlighting** | Full highlighting for Groovy and PowerShell |
| **Smart Editing** | Bracket matching, auto-indentation, code folding, multiple cursors |
| **Find & Replace** | <kbd>Ctrl</kbd>+<kbd>F</kbd> — supports regex and case-sensitive matching |
| **Go to Line** | <kbd>Ctrl</kbd>+<kbd>G</kbd> — jump to specific line numbers |

## Output Panel

View script execution results with parsed output and validation.

<img src="/lmda-composer/getting-started/interface-output.png" alt="LMDA Composer output panel" style="border-radius: 0.5rem; margin: 1rem 0;" />

<Tabs>
  <TabItem label="Raw Output" icon="document">
    Complete text output exactly as returned by the collector. Preserves formatting and line breaks.
  </TabItem>
  <TabItem label="Parsed" icon="list-format">
    Structured view based on mode:
    - **AD Mode** — Discovered instances in a sortable table
    - **Collection** — Datapoints with numeric values
    - **Batch** — Grouped by instance (wildvalue)
  </TabItem>
  <TabItem label="Validation" icon="approve-check-circle">
    Real-time validation against mode-specific rules:
    - Green checkmark = All output valid
    - Yellow warning = Warnings present
    - Red X = Errors found
  </TabItem>
  <TabItem label="Graph" icon="random">
    For TopologySource scripts, displays a visual network graph of discovered topology.
  </TabItem>
</Tabs>

### Header Controls

The output panel header includes:
- **Execution Status** — Success, error, or running indicator
- **Duration** — How long the script took to execute
- **Copy** — Copy output to clipboard
- **Fullscreen** — Expand output panel
- **Clear** — Clear current output

## Right Sidebar

A collapsible panel with additional tools. Toggle with <kbd>Ctrl</kbd>+<kbd>B</kbd> / <kbd>⌘</kbd>+<kbd>B</kbd>.

<img src="/lmda-composer/getting-started/interface-sidebar.png" alt="LMDA Composer right sidebar" style="border-radius: 0.5rem; margin: 1rem 0;" />

<Tabs>
  <TabItem label="Properties" icon="information">
    View device properties when a device is selected:
    - Searchable property list
    - Click property name to copy
    - Grouped by type (system, custom, auto, inherited)
  </TabItem>
  <TabItem label="Snippets" icon="code">
    Access code templates and patterns:
    - Built-in snippets for common tasks
    - Your custom user snippets
    - Filter by language and type
    - Click to insert at cursor
  </TabItem>
  <TabItem label="History" icon="history">
    Review past script executions:
    - Timestamp and duration
    - Success/error status
    - Click to reload script
    - View full execution details
  </TabItem>
</Tabs>

## Status Bar

The bottom bar shows current state at a glance.

**Left Side:**
- Execution status indicator
- Portal connection status
- Collector name (online/offline)
- Selected device name

**Right Side:**
- Current mode badge
- Language badge
- Line and character count
- Help popup with shortcuts

## Welcome Screen

When no tabs are open, the welcome screen provides quick actions organized into three categories:

### Create New
- **New Script** — Create a blank Groovy or PowerShell file
- **New LogicModule** — Launch the guided wizard to create a module (requires portal connection)

### Open Existing
- **Open File** — Load a script from your computer
- **Open Module Folder** — Open a previously saved module directory

### Import from LMX
- **Browse Modules** — Import scripts from your portal's LogicMonitor Exchange

The welcome screen also shows:
- **Recent Files** — Quick access to recently opened files and module directories
- **Workflow Steps** — Getting started guidance for new users
- **Key Features** — Overview of LMDA Composer capabilities

<Aside type="tip" title="Switch to API Explorer">
  Use the button at the bottom of the welcome screen to switch to the API Explorer workspace.
</Aside>

## Keyboard Shortcuts

LMDA Composer uses chord-based shortcuts to avoid conflicts with browser extensions.

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> | Command palette |
| <kbd>Cmd</kbd>+<kbd>Enter</kbd> | Run script |
| <kbd>Cmd</kbd>+<kbd>B</kbd> | Toggle sidebar |
| <kbd>Cmd</kbd>+<kbd>S</kbd> | Save file |
| <kbd>Cmd</kbd>+<kbd>,</kbd> | Open settings |
| <kbd>Cmd</kbd>+<kbd>K</kbd>, <kbd>N</kbd> | New file |
| <kbd>Cmd</kbd>+<kbd>K</kbd>, <kbd>F</kbd> | Open module folder |

<Aside type="note">
  On Windows/Linux, replace <kbd>Cmd</kbd> with <kbd>Ctrl</kbd>.
</Aside>
