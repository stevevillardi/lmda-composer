## Settings Categories

<Tabs>
  <TabItem label="Appearance" icon="sun">
    ### Theme

    | Option | Description |
    |--------|-------------|
    | <Badge text="System" variant="default" /> | Follow OS light/dark setting |
    | <Badge text="Light" variant="note" /> | Always use light theme |
    | <Badge text="Dark" variant="tip" /> | Always use dark theme |

    The theme applies immediately when changed.
  </TabItem>
  
  <TabItem label="Defaults" icon="setting">
    ### Default Language

    Choose the language for new files:
    - **Groovy** (default)
    - **PowerShell**

    ### Default Mode

    Choose the execution mode for new files:
    - **Freeform** (default)
    - **Active Discovery**
    - **Collection**
    - **Batch Collection**
  </TabItem>
  
  <TabItem label="Editor" icon="pencil">
    ### Font Size
    - Range: 10px to 24px
    - Default: **14px**
    - Changes apply to all tabs

    ### Tab Size
    - Range: 2 to 8 spaces
    - Default: **2 spaces**
    - Affects new indentation

    ### Word Wrap
    - **On** (default): Lines wrap within viewport
    - **Off**: Horizontal scroll for long lines

    ### Minimap
    - **Show**: Display minimap on right for quick navigation
    - **Hide** (default): No minimap, more editor space
  </TabItem>
  
  <TabItem label="History" icon="history">
    ### History Size

    Maximum execution history entries to keep:
    - Range: 10 to 500
    - Default: **50**
    - Oldest entries removed when exceeded
  </TabItem>
  
  <TabItem label="API Explorer" icon="seti:json">
    ### API History Limit

    Maximum API requests to store:
    - Range: 10 to 200
    - Default: **10**

    ### Response Size Limit

    Maximum response size to display:
    - Default: **250,000 bytes** (~250KB)
    - Larger responses truncated for performance
    - Full response still available via copy
  </TabItem>
</Tabs>

## Applying Settings

### Immediate Changes

Most settings apply immediately:
- Theme changes instantly
- Font size updates all tabs
- No refresh required

### New File Settings

Some settings affect only new files:
- Default language
- Default mode
- Tab size for new indentation

Existing tabs keep their current settings.

---

## Resetting to Defaults

<Steps>

1. Scroll to bottom of Settings dialog

2. Click **Reset to Defaults**

3. Confirm the action

4. All settings restored to defaults

</Steps>

<Aside type="caution" title="Irreversible">
  Reset cannot be undone. Note your custom settings before resetting.
</Aside>

---

## Settings Storage

### Where Settings Are Stored

Settings persist in:
- Browser local storage
- Per-browser (not synced)
- Survive browser restarts

### Clearing Settings

Settings are cleared when:
- You reset to defaults
- Browser data is cleared
- Extension is uninstalled/reinstalled

## Recommended Settings

<Tabs>
  <TabItem label="Large Monitors" icon="laptop">
    For large/high-resolution displays:
    - **Font Size**: 16-18px
    - **Minimap**: Show
    - **Word Wrap**: Off
  </TabItem>
  <TabItem label="Smaller Screens" icon="seti:favicon">
    For laptops or smaller screens:
    - **Font Size**: 12-14px
    - **Minimap**: Hide
    - **Word Wrap**: On
  </TabItem>
  <TabItem label="PowerShell Dev" icon="seti:powershell">
    For PowerShell development:
    - **Default Language**: PowerShell
    - **Tab Size**: 4 (PowerShell convention)
  </TabItem>
  <TabItem label="Quick Testing" icon="rocket">
    For rapid script testing:
    - **Default Mode**: Freeform
    - **History Size**: 200+ (keep more history)
  </TabItem>
</Tabs>

| Key | Action |
|-----|--------|

| <kbd>Tab</kbd> | Move to next field |
| <kbd>Enter</kbd> | Apply selection |

## Settings vs. Tab State

<CardGrid>
  <FeatureCard title="Settings (Global)" icon="settings" color="blue">
    Configured in Settings dialog. Apply to all sessions, persist across restarts.
  </FeatureCard>
  <FeatureCard title="Tab State (Per-Tab)" icon="document" color="purple">
    Managed per tab: language selection, execution mode, script content.
  </FeatureCard>
  <FeatureCard title="Session State" icon="history" color="green">
    Temporary: open tabs, sidebar state, last used collector.
  </FeatureCard>
</CardGrid>

## Troubleshooting Settings

<CardGrid>
  <FeatureCard title="Settings Not Saving" icon="warning" color="orange">
    Check browser storage isn't full. Verify extension has storage permission. Try clearing and re-setting.
  </FeatureCard>
  <FeatureCard title="Theme Not Applying" icon="sun" color="amber">
    Refresh the page. Check OS theme setting. Try a specific theme instead of System.
  </FeatureCard>
  <FeatureCard title="Editor Changes Not Visible" icon="edit" color="blue">
    Check you're on a script tab. Try switching tabs. Reload the extension page.
  </FeatureCard>
</CardGrid>
